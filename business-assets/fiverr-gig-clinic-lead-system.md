# Fiverr Gig Draft: Clinic Lead Capture System

## Gig title

I will build a lead capture and follow-up system for your clinic website

## Short positioning

I help clinics and service businesses stop losing website enquiries by setting up a simple lead form, Google Sheet CRM tracker, notification path, thank-you page, and follow-up templates.

## Package idea

### Basic — Lead Form + Tracker

- Setup or improve one enquiry form
- Connect to Google Sheet CRM
- Add basic statuses and source tracking
- Delivery: 3 days

### Standard — Clinic Lead System

- Everything in Basic
- Landing page or appointment enquiry section
- Thank-you page
- Follow-up templates
- Basic mobile CTA improvements
- Delivery: 5 days

### Premium — Lead System + Website Improvements

- Everything in Standard
- Up to 3 page/section improvements
- Extra form fields/automation logic
- Handover guide or short loom walkthrough
- Delivery: 7 days

## Buyer requirements

- Website URL
- Website platform (WordPress, Wix, custom, etc.)
- Admin access or collaborator access method
- Preferred notification email
- Services/treatments to capture
- Booking link or phone number if available
- Brand/logo if a new landing section is needed

## FAQ

**Do I need a full CRM?**
No. This starts with a lightweight Google Sheet CRM so enquiries are not missed.

**Can you work on WordPress?**
Yes. This is especially suitable for WordPress clinic and local service websites.

**Can you guarantee more patients?**
No ethical seller should guarantee patient volume. The service improves capture, tracking, and follow-up so fewer enquiries are lost.

**Can you add WhatsApp or booking links?**
Yes, if you provide the correct link/number and approve how it should appear.
